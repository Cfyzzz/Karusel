Инструкция по установке

ПОДГОТОВКА
1. Распковать 
2. Установить Python3 версии

3. Установить pip, если не установлен
sudo apt udate
sudo apt-get install python3-pip

ЧТОБЫ НЕ УСТАНАВЛИВАТЬ ЗАВИСИМОСТИ ПРОЕКТА В СИСТЕМУ, МОЖНО УСТАНОВИТЬ ИХ В ВИРТУАЛЬНОЕ ОКРУЖЕНИЕ
4. Установить виртуальное окружение
sudo pip install virtualenv
5. Создать виртуальное окружение для проекта
sudo virtualenv venv
6. Включить виртуальное окружение
source venv/bin/activate

УСТАНОВКА ЗАВИСИМОСТЕЙ ПРОЕКТА
7. Из папки проекта установить необходимые зависимости
pip install -r requirements.txt
8. Указать нужные настройки сервера в файле settings.py
HOST и PORT

ЗАПУСК ПРОЕКТА
9. Запустить сервер 
python3 app.py

Сервер доступен по адресу HOST:PORT
